WordList is a website to learn a language by adding words and using flash cards to review then.


===== Features =====

- User
    - Create account.
    - Login.
    - Authentication.
    - Token and backend cookies that expires after one week.

- List
    x Create List
    - Rename List
    - Delete List
    x Show all the words inside of lists.
    x show the total of words.
    
- Words
    x Add one or many words at same time.
    x rename words
    x remove words
    x Can be marked as learned if answered right x times.
    x user can reset all the words or just one word.
    x Term will be one word.
    x User can't add a duplicated term.
    x Definitions will be one or more words in a unique string separated by comma.

- WordList
    x WordList will be a list of words separated by x words per list.
    x Review wordList
    x user can restart the flash cards.
    
- Configs
    x Words per wordList.
    x How many times should be answered right until a word be marked as learned.
    x Reset all the learned words.
    x show the total of words, the total of words learned and the total not learned yet.
    
       
===== Interfaces =====

- User
    ID: string;
    email: string;
    password: string;
    
- List
    userID: string;
    ID: string;
    name: string;
    words: Word[];
    
- Word
    term: string;
    definitions: string;
    correctTimes: string;
    learned: string;
